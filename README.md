# Global Store Sales Analysis (Power BI)

Project Overview:-

This project is a Power BI dashboard designed to analyze and visualize global store sales performance.
It provides actionable insights into sales trends, profitability, customer segments, and regional performance using interactive visuals.

The goal of this project is to help stakeholders:

Understand overall business performance

Identify high-performing products and regions

Track key sales and profit metrics

Support data-driven decision making

Tools & Technologies:-

Power BI Desktop

Data Modeling

DAX (Data Analysis Expressions)

Data Visualization

Dataset Description:-

The dataset contains historical sales data for a global retail store, including:

Order details

Customer information

Product categories

Sales, profit, and quantity

Geographic regions

The dataset is included inside the .pbix file.

Key Insights & Features:-

Sales & Profit Overview – Total sales, profit, and order trends

Category & Sub-Category Analysis – Performance by product type

Regional Performance – Sales distribution across regions and countries

Customer Segmentation – Insights by customer segments

Interactive Filters – Dynamic slicers for deeper analysis

Dashboard Highlights:-

Clean and interactive visual design

Drill-down capabilities for detailed insights

KPIs for quick performance tracking

User-friendly layout for business users

How to Use:-

Download or clone this repository

Open the .pbix file using Power BI Desktop

Refresh data if needed

Explore the dashboard using slicers and filters
